js组件

js组件widget.min.js是基于jquery的插件库， 使用简单，通过配置相关参数，能快速开发出界面效果。

使用方法

1. 在页面中引入jquery和widget.min.js文件

2. 按插件要求写好html结构和css样式

3. 调用相应插件方法，配置对应参数即可

系统弹框

options参数列表--$.alert(options)

参数 数据类型 默认值 参数说明
mode string alert

模拟系统弹框采用的展示方式，值为confirm时则有取消和确定两个按钮
wrapClass string alert

模拟弹框最外层框的class，用来控制样式
closeClass string alert-close

弹框关闭按钮的class
confirmClass string alert-confirm

弹框确定按钮的class
promptText string 无

弹框提示的文本内容
closeText string 取消

取消的文本
confirmText string 确定

确定的文本
maskClick boolean false

点击遮罩是否取消
operate function 无

点击确定按钮触发的回调函数

用法


						
调用方法

						
$.alert({

						
　　promptText: '已到最后一道题目'

						
});

						
$.alert({

						
　　mode: 'confirm',

						
　　promptText: '确定重新开始吗？',

						
　　operate: function(){

						
　　　　alert('执行确定后的事件');

						
　　}

						
});

滚动监听

options参数列表--$('.content').monitor(target, options)

参数 数据类型 默认值 参数说明
target(必须) string 无

监听每个内容区的开始位置元素, 标准jquery选择器
options参数
dirWrap string .catalog

监听浮动列表项的父容器, 标准jquery选择器
listWrap string li

监听浮动列表项, 标准jquery选择器
triggerClass string current

当前点击目录的class名字
speed number 400ms

滚动到指定位置时的速度
disTop object 0, 9999

距离顶部多少距离时, 隐藏快速导航目录, 包含两个属性:
minTop: 0(默认), maxTop: 9999(默认)
topDiff number 0

顶部差距, 默认为0. 当浏览器顶部有跟随的浮动层时, 请指定此项的值。
opacity boolean false

快速导航目录显示时, 是否具有透明度效果
animate boolean true

滚动到指定位置时, 是否具有动画效果

用法


						
html结构

						
<div class="container">

						
　　<h2"></h2>

						
</div>

						
<ul class="anchor">

						
　　<li></li>

						
</ul>

						
调用方法

　　dirWrap: '.anchor',

						
　　triggerClass: 'cur'

						
});

圆形进度条动画

options参数列表--$('.circle').circleProgress(options)

参数 数据类型 默认值 参数说明
value number 0

这是唯一一个必填参数。值从0.0到1.0
size number 100

canvas的大小，单位像素
startAngle Math.PI

初始角度
reverse boolean false

是否反向绘制圆弧和动画
thickness auto

进度条圆弧的宽度。默认它自动为size的1/14大小，你可以设置你需要的值。
lineCap string butt

圆弧的线头样式："butt"、"round"和"square"。
fill object { gradient: ["#3aeabb", "#fdd250"] }

圆弧填充的配置。
-{ color: "#ff1e41" }
-{ color: 'rgba(255, 255, 255, .3)' }
-{ gradient: ["red", "green", "blue"] }
-{ gradient: [["red", .2], ["green", .3], ["blue", .8]] }
-{ gradient: [ ... ], gradientAngle: Math.PI / 4 }
-{ gradient: [ ... ], gradientDirection: [x0, y0, x1, y1] }
-{ image: "http://i.imgur.com/pT0i89v.png" }
-{ image: imageInstance }
-{ color: "lime", image: "http://i.imgur.com/pT0i89v.png" }

emptyFill string rgba(0, 0, 0, .1)

空圆弧的颜色。
animation { duration: 1200, easing: "circleProgressEase" }

动画配置。可以参考jQuery animations。你可以设置为false来禁止动画。"circleProgressEase"是一个ease-in-out-cubic easing动画效果
animationStartValue number 0.0

默认进度条动画会在0.0开始，结束与value处。调用该参数可以直接动画。如果需要制作反向动画就将animationStartValue的值设置为1.0。你可以指定0.0到1.0之间的任何数值。
事件 处理程序
circle-animation-start： function(event)-event - jQuery事件
circle-animation-progress function(event, animationProgress, stepValue)：
-event - jQuery事件
animationProgress - 从0.0到1.0
stepValue - 当前的步长值，从0.0到value
circle-animation-end function(event)：-event - jQuery事件

其他


						
如果进度条组件已经被初始化，你可以获取<canvas>：

						
$('#circle').circleProgress({ value: 0.5 });

						
var canvas = $('#circle').circleProgress('widget'); 

						
你可以获取CircleProgress对象实例：

						
var instance = $('#circle').data('circle-progress'); 

						
如果进度条组件已经被初始化，你可以重新绘制已经存在的圆环：

						
$('#circle').circleProgress({ value: 0.5, fill: { color: 'orange' }});

						
$('#circle').circleProgress('redraw');

						
$('#circle').circleProgress();

						
$('#circle').circleProgress({ size: 150 });

						
你可以改变默认的参数配置：

						
$.circleProgress.defaults.size = 50;

用法


						
html结构

						
<div id="circle"></div>

						
调用方法

						
$('#circle').circleProgress({

						
　　value: 0.75,

						
　　size: 80,

						
　　fill: {

						
　　　　gradient: ["red", "orange"]

						
　　}

						
});

						
示例

options参数列表--$('.btn').print(options)

参数 数据类型 默认值 参数说明
globalStyles boolean true

是否包含父文档的样式
mediaPrint boolean false

是否包含media='print'的链接标签。会被globalStyles选项覆盖
stylesheet URL-string null

外部样式表的URL地址
noPrintSelector 任何可用的jQuery选择器 ".no-print"

不想打印的元素的jQuery选择器
iframe 任何可用的jQuery选择器或Boolean 默认true，如果没有iframe选择器被传入会创建一个隐藏的iframe

是否使用一个iframe来替代打印表单的弹出窗口
append/prepend boolean null

是否将用户更新的表单输入框内容作为打印内容（通过迭代每一个表单元素来实现）
deferred 任何可用的jQuery.Deferred对象 $.Deferred()

当打印函数被调用时的jQuery.Deferred对象

用法


						
调用方法

						
$(".btn").print({

						
　　globalStyles: true

						
});
